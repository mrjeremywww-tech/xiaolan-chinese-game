# 小嵐山中文測驗 🐱✍️

一個可愛的卡通風格中文學習遊戲，適合小學一年級學生。

## 🎮 遊戲內容

### 六個測驗關卡：

1. **🔍 古文字猜猜看** - 象形文字識別（山、水、火、木、人、口等）
2. **🧩 組合漢字** - 部首組合遊戲（門+口=問、日+月=明等）
3. **📏 量詞選擇** - 選擇正確的量詞（一朵花、一本書等）
4. **⚖️ 反義詞填空** - 找出相反意思的詞（危險=安全等）
5. **📝 句子排序** - 排列正確的句子順序
6. **✍️ 描紅練習** - 使用 Hanzi Writer 跟著寫字

## 🐱 小嵐山

遊戲的吉祥物，一隻可愛的橘色小貓，會在遊戲中給予鼓勵和提示！

## 🛠️ 技術棧

- HTML5 + CSS3 + Vanilla JavaScript
- [Hanzi Writer](https://hanziwriter.org/) - 中文筆順動畫庫
- Noto Sans TC 字體
- 響應式設計（手機/平板/電腦）

## 📚 內容來源

基於澳門聖公會中學一年級中文工作紙內容設計

## 🎨 風格

- 卡通風格（Cartoon Style）
- 暖色調配色（橙色、藍色、綠色）
- 圓角設計、陰影效果
- 動畫互動回饋
